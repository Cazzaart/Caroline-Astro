Here is demonstration using our custom image component. The following lines of code will not appear in the front end.

First we need to import the general image component, being sure to keep empty lines before an after the following code:

import Image from "../../components/Image.astro";

Then we create a specific image reference being sure to give it a unique name:

import myFirstImage from "/src/assets/placeholder-hd.webp";

Then we can use use the image component with our specific reference like so:

<Image path={myFirstImage} alt="Image alt text goes here"/>

Now you should see the image in the front-end.

To make sure we meet accessiblity requirements the alt tag is a included as part of the image component. Also, if by chance you want the image to also link somewhere, just include link="..." as in the following example.

Here, to add another image we need to create another unique reference, eg:

import mySecondImage from "/src/assets/placeholder-square.webp";

And then:

<Image path={mySecondImage} alt="Image alt text goes here" link="https://astro-sky.vercel.app/"/>

Now in the front end you should see an image with a link.
